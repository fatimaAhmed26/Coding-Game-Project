### How to Play
1. Enter the the game using the link above
2. start solving the questions
3. you can see your score up next to the game tittle 
4. when you are done with the game a page will pop up you can see your score or restart the game 
### Installation
- No installation required! Simply clone the repository to your local machine and open the `index.html` file in your favorite browser to start playing.

### Technologies Used

- **HTML**: For structuring the game content.
- **CSS**: For styling and animating the game interface.
- **JavaScript**: For game logic and interactivity.
